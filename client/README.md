# Client - ReactJS Frontend Microservice

> Frontend microservice for the Tenant Management Service

## Description

This directory will contain the ReactJS frontend application for the tenant management service.

## Prerequisites

- Node.js 16+
- npm or yarn

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build

# Run tests
npm test
```

## Project Structure

```
client/
├── public/          # Static assets
├── src/             # Source code
│   ├── components/  # React components
│   ├── services/    # API services
│   ├── utils/       # Utility functions
│   └── App.js       # Main application
├── package.json     # Dependencies
└── README.md        # This file
```
