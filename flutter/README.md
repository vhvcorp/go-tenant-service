# Flutter - Mobile Application

> Mobile application for the Tenant Management Service

## Description

This directory will contain the Flutter mobile application for the tenant management service.

## Prerequisites

- Flutter 3.0+
- Dart 2.17+
- Android SDK (for Android development)
- Xcode (for iOS development on macOS)

## Getting Started

```bash
# Install dependencies
flutter pub get

# Run on connected device
flutter run

# Build for Android
flutter build apk

# Build for iOS
flutter build ios

# Run tests
flutter test
```

## Project Structure

```
flutter/
├── android/         # Android-specific code
├── ios/             # iOS-specific code
├── lib/             # Dart source code
│   ├── models/      # Data models
│   ├── screens/     # UI screens
│   ├── services/    # API services
│   ├── widgets/     # Reusable widgets
│   └── main.dart    # Application entry point
├── test/            # Tests
├── pubspec.yaml     # Dependencies
└── README.md        # This file
```
